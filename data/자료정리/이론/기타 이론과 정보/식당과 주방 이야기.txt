주방의 형태 및 종류


주방의 형태
전통형 주방 :
식재료를 구매하여 조리전의 준비와 음식을 만드는 장소가 같은 장소에서 이루어진다

혼합형 주방:
조리전의 식재료 준비와 음식을 만드는 장소가 같은 공간에서 이루어지지만, 서로 구획
을 구분하고 있다

분리형 주방 :
조리 전의 식재료 준비와 조리업무를 수행하는 공간이 서로 공간적으로 떨어져서 운영
된다

편의형 주방 :
완전 또는 반가공된 식재료만을 구입하기 때문에 마무리 주방만 있으면 되는 형태이다


주방의 종류
중심지원 주방(central kitchen) :
육수, 소스, 드레싱, 육류, 어패류의 정선작업 등 다른 영업주방에서 이용하는 기본적인
식재료를 먼저 처리하는 곳으로 규모가 크고 세분화 되어 있다
- 더운요리 주방(hot kitchen)
- 찬요리 주방(cold kitchen)
- 어육가공 주방(butcher)
- 제과·제빵 주방(bakery kitchen)
- 기물세척 주방(steward)

영업 주방 :
음식을 만드는 주방과 판매할 수 있는 홀이 있는 주방의 형태로 메인 주방에서 일차적으
로 만들어진 육수, 드레싱, 고기, 생선, 디저트 등을 제공받아 고객의 주문에 따라 완성,
판매하는 주방이다




주방의 직무

- 기준이 명확하지 않고 겹치는 부분이 있을 수 있음(참고용)

총괄 주방장(Executive Chef) :
운영자로서 주방을 관리함
- 사장, 오너 쉐프, 점장, 마스터, 쉐프
- 보통 10년 이상의 경력

주방장(Chef de Cuisine / Kitchen Chef) :
실질적 주방장
운영적인 측면에도 관여하면서 실무를 책임진다
- 수석 매니져, 점장, 마스터, 쉐프, 차장
- 보통 7년~10년 이상의 경력

부주방장(Sous Chef / Deputy Kitchen Chef / 2nd Chef) :
주방장을 도와 주방 업무 전체를 관리한다
- 매니져, 관리자, 과장
- 보통 5년~7년 이상의 경력

파트장(Chef de partie / Line Chef) :
각 파트의 책임자
- 캡틴, 매니져, 대리
- 보통 3년~5년 이상의 경력
- 소시에, 로티셔, 푸아숑니에, 앙트루메티에, 가드망제, 파티셰

주임 조리사(Demi chef de partie):
파트의 조리와 관리를 맡은 조리사
- 트레이너, 주임
- 보통 2년~3년 이상의 경력

조리보조(Commis / Junior Chef) :
조리 보조, 청소, 잡일, 시키는 일
- 꼬미, 막내, 트레이니
- 보통 0년~2년 이상의 경력


전담 업무에 따른 호칭

아브와이르(Aboyeur / Barker) :
주방과 홀 사이에서 오더를 하거나 음식이 나갈 수 있도록 함
- 백 매니져 혹은 백 스테이션 근무자

코뮈나르(Communard) :
직원들이 먹을 식사를 만든다
- 식사장

소시에(Saucier) :
소스류와 간단한 가니시 또는 소테(볶거나 굽는) 요리를 맡는다

로티셔(Rotisseur) :
고기를 굽거나 삶는 요리를 맡는다
- 그릴아딘, 프라이터리
- 그릴아딘(Grillardin) : 그릴에서 불에 구운 고기 요리를 맡는다
- 프라이터리(Friturier) : 기름에 튀기는요리를 맡는다

푸아숑니에(Poissonier) :
생선의 손질, 요리를 담당한다

에까이외희(Ecailler) :
갑각류, 어패류 따위를 손질하고 요리한다

앙트루메티에(Entremetier) :
전채요리 앙트레(엔트리)를 조리한다
보통 수프, 달걀, 익힌 채소 요리를 맡는다
- 포타져, 레구미에
- 포타져(Potager) : 수프, 스튜, 스톡(육수) 등을 만든다
- 레구미에(Legumier) : 채소를 손질하고 조리한다. 간단한 가열 조리를 하기도 한다

가드망제(Garde Manger) :
육류, 육가공 식품, 샐러드 등 차가운 재료를 손질하고 가공하고 조리한다
- 부쳐, 차커티에
- 부쳐(Boucher) : 고기와 가금류를 도살, 손질한다
- 차커티에(Charcutier) : 햄이나 소시지 등 육가공품을 제조하고 관리한다

파티셰(Patisseur / Patissier) :
디저트 키친의 관리자
디저트, 사탕, 초콜릿 등을 만들고 관리한다
- 글라시에, 블랑저
- 글라시에(Glacier) : 아이스크림을 만든다
- 블랑저(Boulanger) : 빵, 과자 등 오븐 제과 제빵을 담당한다

플런저(Plongeur or Escuelerie) :
설거지를 하는 직원

노예(Stagie / Stagiare) :
자발적 노예(일을 배우기 위해)
돈을 받지 않고 일 함
미슐랭 매장에 많음
