단위와 차원

우리들이 일상 생활에서 사용하는 물리량에는 길이, 질량, 시간, 온도, 압력, 등이 있는데
이들을 표기하는 방법을 단위라 한다
물리량 = 숫자 * 단위 ex 10km
단위는 나라, 지역에 따라 다르게 사용하며 가장 널리 사용되는 것이 
CGS 단위계, MKS 단위계, English 단위계, FPS 단위계이다
CGS 단위계 :  Centimeter, Gram, Secnod
MKS 단위계: Meter, Kilgram, Second
English 단위계 : Inch(in.), Feet(ft), pound-mass(lbm), Second, Farenheit(ºF)
자연과학에서는 CGS나 MKS를 주로 사용해왔으며
1960년 프랑스 파리에서 개최된 국제도량형 위원회에서 SI 단위계를 통일하여 사용하기로 합의하였다
가장 기본이 되는 물리량은 일반적으로 질량, 길이, 시간, 온도를 사용하며
이들 기본량을 기본 차원(Fundamental Dimension)이라 한다
질량 : M(mass), 길이 : L(length), 시간 : t(time), 온도 : T(temperature)
기본 차원의 기본량을 측정하는 단위를 기본 단위라 하며 - 표 1-1
그 외 많은 물리량은 기본 물리량의 제곱이나 조합으로 나타낼 수 있다 = 유도단위

SI 단위계 : - 표 1-2
길이 : meter, 시간 : second, 질량 : kilogram, 온도 : kelvin을 사용하기로 정한 단위계
SI 단위계에는 10진법의 접두어가 붙는다
접두어는 10³배수의 사용을 권장한다 :
10^-6 (mirco (μ)), 10^-3(milli(m)), 10³(kilo(k)), 10^6(mega(M))와
10³의 배수가 아닌 10²(heto(h), 10¹(deca(da)), 10^-1(deci(d)), 10^-2(centi(c)) 등이 있다


중요 물리량

면적 : 두 길이의 곱 : square meter = m²

부피 : 세 길이의 곱 : m³ = 10³L = 1000L, 1L = 10cm³
- 미국, 유럽에서는 gallon을 많이 사용하는데, 미국과 유럽의 gallon의 부피가 다르다
- 1 gal = 4.546L, 1 USgal = 3.785L
- 유체의 유량은 파이프 단면적A(m²) * 평균유속v(m/s) 이다 - 그림 1-1
-- 체적유량 Q(m³/s) = A(m²) * v(m/s) 

밀도 : 단위부피당 질량 kg/m³
- 밀도 p ≡ 질량(kg)/부피(m³) ≡ m(kg)/V(cm³)
- 고체와 액체의 밀도는 압력에는 거의 영향을 받지 않지만 온도에 따라서는 약간 변한다
- ex. 0ºC에서 물의 밀도는 0.99987g/cm³ 이다
- 기체의 밀도는 액체보다 훨씬 작으며 온도와 압력에 따라 크게 변한다
- 밀도 대신 비중을 사용하기도 하는데 비중은 물의 밀도에 대한 비교로
  물질의 밀도 / 물의 밀도 이다 (4ºC의 물의 밀도를 1로 둔다)
- 고체 밀도 : - 표 1-3
-- 물체중의 모든 빈 공간을 제외한 고체 성분만의 밀도
-- 물의 밀도 : 1000kg/m³, 고염도, 고지방 식품을 제외한 대부분 식품성분의 고체밀도는 1400 ~ 1600kg/m³ 정도이다
- 입자 밀도 : - 표 1-4
-- 개별 입자(ex. 사과 1알, 쌀 1톨)의 질량을 입자의 부피로 나눈 것
-- 입자에 존재하는 세공(pore = 아주 작은 구멍)을 포함한 질량과 부피를 측정한다
-- 대부부느이 신선재료(채소, 과일, 어류, 육류)는 수분을 많이 포함하고 있어 물의 밀도인 1000kg/m³를 크게 벗어나지 않는다
- 부피밀도 : - 표 1-5
-- 용기에 식품을 넣고 질량을 측정하여 계산한다
-- 무피밀도 ≡ 질량/벌크 부피
-- 식품의 공극(pore space)률 : (1-부피밀도/고체밀도) * 100%
-- 입자간 공극률 : (1-부피밀도/입자밀도) * 100%

속도 및 가속도 :
- 속도(v)는 이동한 거리/시간 변화량이다 : v ≡ Δx/Δt	단위 : m/s
- 가속도(a) 는 시간에 따른 속도의 변화량이다 : a ≡ dv/dt	단위 : m/s²

힘과 무게 :
- 힘 : 질량 m을 가속도 a로 가속시키는 것 힘 : F= ma 	단위 kg * m/s² 
- 간단하게 사용하기 위해 단위 N(newton)을 사용한다 : N = 1kg * 1m/s²
- 중력 : 물체에 무게를 가하는 지구의 인력
-- 물체가 자유낙하시 중력가속도로 낙하한다. 따라서 무게 W ≡ 질량 m * g = mg
-- 위도 45º 평균 해수면에서의 중력가속도를 표준 중력가속도라 한다
-- 표준 중력 가속도 g = 9.80665 m/s²

몰 : 
자연과학에서는 개수를 단위로 설정하지 않는다 
그러나 예외적으로 원자 또는 분자가 6.02* 10^23개 집합하여 있을 때 이것에 mol의 단위를 붙인다
ex. 탄소원자 1mol = 12g
탄소 원자 6.02 * 10^23개는 12g이며 이것은 탄소의 원자량이다
원자 1mol의 /질량을 g 단위로 나타낸 것을 원자량이라 한다
또는 분자 6*10^23개 또한 1mol 이며 이것의 질량을 분자량이라 한다
- 질량 분율 및 몰 분율 :
- 여러 종류의 화합물로 구성된 혼합물에서 어떤 물질이 얼마나, 어떤 비율로 들어있는지(조성)을 아는 것은 매우 중요하다
- 조성은 농도, 분율로 나타낼 수 있다
- A + B 의 혼합물에서의 분율 :
-- A의 질량의 분율 ≡ A의 질량/총 질량 
-- A의 몰 분율 ≡ A의 몰 수/ 총 몰 수
- 농도 : 
- 용액 중 어떤 성분의 농도를 나타내는데는 몰 농도를 많이 사용한다
- 몰 농도 M의 단위 : mol/L
-- A의 몰 농도 M ≡ A의 몰 수/용액의 부피 
- 농도를 질량 농도로 나타내기도 한다
-- 질량 농도의 단위 : kg/m³ ≡ A의 질량/혼합물의 부피
- 미량 성분의 농도를 ppm, ppb로 표현한다
-- ppm : parts per million = 1/1000000
--- 액체에서의 단위 : gram = 1g에 1/1000000g 포함 = 1톤에 1g 포함
--- 기체에서의 단위 : mol = 1mol에 1/1000000mol 포함 = 1000000mol에 1mol 포함
--ppb : parts per billion = 1/1000000000 = 1/10억

온도 : - 그림 1-2
물질이 가진 열 에너지를 숫자로 나타낸 것
- 수은 온도계 : 수은이 온도가 올라감에 따라 부피가 팽창하는 원리를 이용
- 저항 온도계 : = 전자 온도계. 도체의 전기 저항이 온도에 따라 변하는 성질을 이용
일반적으로 사용하는 단위는 섭씨 이다
- 섭씨(Celsius scale)  : 1atm(1기압 = 평균기압 - 기타 참고 자료 - 압력의 단위 참고)에서 
  물의 어는점(0ºC)과 끓는점(100ºC)를 100등분 하여 표기한 것
- 화씨(Fehrenheit scale) : 물이 어는 점과 끓는 점을 180등분한 후
  소금물(소금 1 : 물 3, 정확히는 농도 24.8%의 소금물)이 어는 점을 0ºF로 설정하여 표기한 것
- 켈빈(Kelvin) : 온도가 내려갈수록 분자의 운동 에너지가 감소하다가 분자가 정지하게 되는 점(절대 0도(absolute zero point))을
  섭씨 기준으로 측정하면 -273.16ºC 이다. 이 절대 0도를 0K로 설정하여 표기한 것
-- 섭씨의 값은 켈빈 값 + 273.16이 된다
-- 섭씨와 켈빈은 단위를 구분하는 기준이 동일하므로 켈빈의 변화량은 섭씨의 변화량과 같다

일과 에너지 및 동력 :
- 일 : 힘이 작용하여 일정한 거리 만큼 물체가 이동한 것
- 정확한 의미로 기계적인 일은 힘이 작용하여 힘의 방향으로 움직인 거리(x)와 힘(F)의 곱이다
-- 일 W ≡ Fx 	단위 : J
-- 1J = 1N * 1m 	- 표 1-6
- 에너지 : 일을 할 수 있는 능력	∴일과 같은 단위를 가진다
- 운동에너지, 위치에너지, 내부에너지, 열에너지 등이 있으며
  일, 에너지, 열의 단위는 SI 단위계에서 모두 J이다
  CGS 단위계에서는 cal(calorie)를 사용하며 이는 물 1g의 온도 1ºC 올리는데 필요한 열량이다
-- 1cal ≡ 4.184J
- 동력 : power. 일을 하는 속도 또는 에너지를 소비하는 속도
- 단위는 SI 단위계에서 watt(W)를 사용하며 1W = 1초동안 1J의 일을 하는 것을 의미한다
- watt ≡ joule/secone 또는 W ≡ J/s



기체와 증기

기체 거동의 가장 큰 특징은 밀도가 온도와 압력에 따라 크게 변한다는 것이다 - 
온도(T), 압력(P), 부피(V)의 상관관계를 이해하는 것이 중요하다

이상기체 법칙 : - 그림 1-3
기체 분자가 매우 작은 부피를 가지며 분자 상호간 아무런 힘이 작용하지 않는 단단한 구로서 
용기 벽면에 탄성적으로 충돌된다고 가정하자
이같은 성질의 기체를 이상기체(ideal gas)라고 한다 

기체중의 분자는 용기 속에 있을 때 열운동을 하면서 용기 벽에 충돌하는데
이 충돌에 의하여 용기 벽의 단위면적에 미치는 힘이 압력이다
따라서 분자의 개수를 증가시키거나 온도를 높여 분자의 열운동을 활발하게 하면
분자가 용기 벽에 충돌하는 횟수가 증가해 압력이 증가한다
기체의 압력 p는 기체의 농도 c(mol/m³)와 온도 T에 비례한다 : p∝cT     - 그림 1-4, 그림 1-5

표준 조건(0ºC, 1atm)에서 이상기체 1mol의 부피는 22.414L이다
1L의 에탄올은 1기압 100ºC에서 기체로 만들면 부피가 526L가 된다(526배가 된다)
이처럼 기체 연료는 부피가 매우 크기 때문에 가압하여 부피를 작게 만들거나 액화시키는 것이 좋다
가스회사가 연료를 다룰때나 고압의 환경으로 기체를 다룰 때는 내압성이 우수한 공 모양의 탱크나 용기를 사용한다

이상기체의 혼합물 :
2종류 이상의 이상기체 A와 B를 용기에 넣었을 때의 압력은 A의 압력과 B의 압력의 평균이 아닌
A의 분압과 B의 분압의 합이다 : P=p(A) + p(B)
고산 등반 시 산소통을 사용하는 이유는 고도 5.5km에서 공기의 전압이 1/2로 감소하는데
이 때 산소의 분압도 1/2로 낮아져 호흡 시 산소가 평소의 1/2밖에 들어오지 않기 때문이다


기체 - 액체계에서 포화증기의 성질 

기체 : 정상적인 조건에서 기체의 형태로만 존재
증기 : 정상적인 상태에서 액체인 물질이 기체의 형태를 한 것
- 증기는 기체 상태이지만 액체로도 존재할 수 있는 성분
증발 : 액체 상태의 물 분자가 공간으로 날아가 증기가 되는 현상
응축 : 공간중의 물 분자가 벽면에 충돌하여 일부 물 분자가 액체 상태로 되돌아가는 것

특정 공간에서 증발이 계속됨에 따라 기체의 물 분자 수가 점점 많아지면서
증발과 응축의 속도가 같아지는 평형상태에 도달하게 된다
이와 같은 상태를 상부 공간은 중기로 포화되었다고 표현한다 - 그림 1-8
포화증기의 압력인 포화증기압은 물의 양에는 영향을 받지 않고 온도에 따라 좌우된다 - 그림 1-9
포화증기는 이상기체와 같은 성질을 가진다
즉 다른 기체의 분압에 영향을 받지 않고 독립적인 분압을 갖는다
액체의 포화증기압이 공기의 전압과 같아질 때 액체는 끓기 시작한다 - 그림 1-10
- 액체의 포화증기압과 기압의 데이터는 기압에 따른 액체의  끓는점을 알 수 있게 해준다
