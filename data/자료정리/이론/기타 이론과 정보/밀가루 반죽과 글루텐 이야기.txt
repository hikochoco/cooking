밀가루 반죽과 글루텐


밀가루에 물을 넣고 물리력을 가해 반죽을 하면 글루텐이 형성된다

글루텐 :
점성을 높이며 가소성이 높아진다
- 가소성 : 탄력과 변형된 상태를 유지하려는 속성 <->탄성
글루텐은 자연상태에는 존재하지 않고 물과 밀가루가 만나야 생성되는 물질이다
밀가루 단백질의 80%는 글루테닌 + 글리아딘이며
이 두 단백질이 결합해 글루텐이 된다
글루텐의 그물조직은 기체와 수분을 잡아두며 이 때문에 반죽을 가열하면 부풀게 된다


밀가루별 단백질 함량 

강력분 : 11% 이상 - 제빵에 주로 사용 - 쫄깃
중력분 : 9~11% - 다용도, 제면에 사용 - 중간
박력분 = 7~9% - 제과에 주로 사용 - 바삭
기타. 듀럼밀 : 13%이상. - 파스타 - 쫄깃 + 단단


글루텐과 외부 요인

밀가루의 종류 : 
단백질의 함량과 글루테닌과  글리아딘의 비율
- 단백질 함량이 높으면 글루텐 형성이 오래 걸리지만 생성량과 유지력이 좋다
- 글리아딘의 성질 : 점착성과 연성
- 글루테닌의 성질 : 견고성
- 글리아딘의 비율이 높으면 글루텐 형성이 느리지만 오래 지속된다(경질밀)
- 글루테닌의 비율이 높으면 글루텐이 빨리 형성되지만 빨리 약화된다(연질밀)

물 :
글루텐 형성에 필수이며 단백질 함량이 높은 밀일수록 많은 물이 필요하고
너무 많아도 글루텐이 잘 만들어지지 않는다

물리력 :
글루텐 형성에 필수적이며 글루텐 조직을 그물망처럼 만들고 기체를 유입시키나
너무 강한 물리력으로 오래 반죽하면 글루텐 섬유가 끊어져 탄력, 가소성, 연성, 점착력이 약해진다
- 글루텐이 거의 없고 가열시 잘 부풀지 않아 너무 질기거나 단단해질 수 있고 쫄깃함을 잃을 수 있다 (글루텐이 뭉개지는 느낌)

가루의 입도 :
가루의 입자가 작을수록 글루텐이 잘 형성된다

효소 :
글루텐을 작은 조각으로 부수어 만들어 부드럽고 신축성을 갖게 한다
- 전분과 단백질을 분해시켜 전분은 설탕으로 바뀌고 단백질은 글루텐으로 재결성 된다
- 이 과정에서 프로테아제라고 불리는 단백질 분해 효소가 글루텐을 부드럽게 해 본 반죽의 신장성을 증대시킨다

발효 :
탄산가스와 알코올이 생성되어 반죽을 부풀게 하며 산도를 높여
글루텐을 강화해 점착력, 탄성이 강해지고 부피가 커지고 식감이 부드러워진다
과다발효는 에틸알코올이 글리아딘을 녹이고 부피가 과다하게 커져 탄력을 잃게 할 수 있다

pH : 
이상적인 pH는 5~6이다
베이킹 소다는 pH 수치를 높여 산성을 낮춰 팽창성이 커지고 기공이 증가하며 반죽을 부드럽게 한다

소금 : 
보통 밀의 1.5~2%를 첨가한다
글루텐을 강화한다
특히 글리아딘을 강화하여 반죽이 더 부드럽고 쫄깃해지며 지속력이 좋아진다
너무 많이 넣으면 수분을 빨아가 글루텐 형성을 방해한다
대부분의 염류는 수소 결합에 크게 영향을 미치기 때문에 전분의 호화를 촉진시킨다(면의 경우 면이 빨리 익음)

설탕 :
물을 흡수하여 글루텐 형성에 필요한 물이 부족해져 글루텐 형성을 방해한다
- 바삭함을 더 극대화할 수 있다
- 쫄깃함을 유지하면서 단순히 단 맛을 추가하려면 충분히 반죽 후 설탕을 추가한다

유지류 : 
글루텐에 필요한 물 흡수를 방해하여 글루텐 형성을 방해한다
하지만 글루텐이 충분히 형성된 반죽에 유지를 추가하면 반죽이 부드러워질 수 있다

식초 : 
글루테닌을 녹여 글루텐 형성을 방해한다
또는 생성된 글루텐을 약화시킨다

알코올 : 
글리아딘을 녹여 글루텐 형성을 방해한다
또는 생성된 글루텐을 약화시킨다

방치(숙성 또는 휴지) : - 오토리즈 = autolyse = autolysis = 자기분해 - 반죽 후 보통 20분~ 1시간 정도 휴지. 또는 그 이상
반죽 직후는 반죽의 글루텐이 팽팽하게 당겨져 있는 상태이다
반죽을 휴지시키면 균질화되고 글루텐이 안정화고 느슨해져 부드러워지고 성형하기 쉬워진다
탄성이 줄어드는 시점은 강력분 반죽이 60분, 중력분이 45분, 박력분이 15분 정도이다
- 효소가 단백질과 전분을 분해하고 성질을 바꾸어 글루텐을 재결성해 부드럽게 하고 잘 늘어나게 한다
- 밀가루가 물을 충분히 흡수하여 수화하여 가열조리 후 촉촉한 상태를 유지하며 노화를 늦춘다
- 산소와의 마찰을 줄여 맛과 색, 표면, 향이 좋은 결과물이 나온다

우유 : 
수분 보유 능력이 우수하여 발효의 안정성이 높아진다

계란 :
난황의 유화제 성질이 지방을 고르게 분산시켜 부드러워지며 팽창성이 좋아지고 색깔을 낸다
난백의 단백질이 반죽의 단백질을 강호한다

온도 :
온도가 높으면 글루텐이 잘 형성된다(21~27ºC 정도)
온도가 낮으면 글루텐이 잘 형성되지 않는다


글루텐이 강하다는 것은 탄력과 신장성이 뛰어나 전분을 든든하게 에워싸고 있다는 뜻이다
그리고 이산화탄소를 잘 모아 볼륨감이 잘드러나게 된다
글루텐이 약하다는 것은 쫄깃한 밀가루 음식이 요구하는 이 같은 특성을 유지할 수 없다는 뜻이다

무조건 글루텐이 강하고 많아야 좋은 것은 아니다
음식의 종류에 따라 개인별 기호에 따라 먹는 습관에 따라 그 선택폭이 다양하다
부드러운 면을 만들 때는 중력분이 좋고
바삭한 빵을 원할 경우 박력분과 같은 케이크에 사용되는 밀이 좋으며  
쫄깃하고 빵빵한 빵을 원할경우 강력분과 같은 밀이 사용된다
프랑스 대표빵인 바게트는 중력분이 사용되는 것처럼 선택의 폭이 넓다
