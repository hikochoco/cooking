피자 

최초의 피자 :
1830년도에 화산의 열기로 구워먹은 빵이라는 기록이 있음

피자를 가장 많이 소비하는 국가는 미국이다

사실 대단한 음식이 아니라 
그저 빵 위에 토마토, 치즈, 바질이 들어간 서민 음식이었다

마르게리따 :
토마토, 바질, 치즈 - 이탈리아의 색깔
마르게리따 - 여왕의 이름

피자의 기준:
associaione VERACE PIZZA NAPOLETANA (베라 피자 나폴리 연맹)
기준이 확실한 이탈리아의 인증. 가장 권위 있음
- 피자 마리나라 : 토마토, 마늘 오레가노, 올리브 오일만 재료로 사용 가능
- 피자 마르게리따 : 토마토 모짜렐라 바질, 올리브 오일만 재료로 사용 가능
- 피자 마르게리따 엑스트라 : 마르게리따 + 특정 치즈 (DOP 인증 치즈)
  DOP인증 : 감빠냐에서 나온 버팔로 모짜렐라 치즈
- TIPP"00" 밀가루만 사용
- 산 마르짜노(SAN MARZANO)에서 나온 토마토만 사용
- 우유와 물소젖(버팔로) 모짜렐라만 사용 가능
- 장작 오븐(화덕)만 사용 가능
- 손반죽만 사용 가능


세계의 피자 

시실리아 피자 :
빵이 두껍고 토핑이 많다
단맛이 많이 난다
빵 위에 치즈, 소스, 토핑 순서로 올린다 - 빵이 젖지 않게 하기 위해

시카고 피자 :
PIZZA UNO가 원조
빵, 치즈, 토핑, 소스 순서로 올린다

뉴욕 스타일 피자 :
석쇠에 구운 큰 피자. 접어먹는다(닌자거북이)
크러쉬페퍼, 스리라차, 파마산 가루치즈, 타바스코, 오레가노 등의 컨디먼트들을 같이 나오기 시작

디트로이트 피자 : 
각진 철판에 구운 피자
brick(벽돌 모양)치즈 사용

캘리포니아 스타일 피자 : 
토핑이 굉장히 풍부하고 다양하다(피자헛, 도미노 스타일)
울프강 퍽이라는 요식업 사업가가 투자하여 창업한
캘리포니아 피자 키친이 유명

세인트 루이스 스타일 피자 :
발효가 안된 바삭한 얇은 피자
단맛이 강하고 오레가노가 많이 들어간다
파티컷(사각형 컷팅)으로 자른다
프로벨 치즈(가공 치즈)사용
