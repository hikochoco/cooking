마요네즈

물에 떠있는 유화된 지방 미립자에 맛을 더한 것 - 수중유적형 교질 미립자

마요네즈의 소향의 기름은 빛을 많이 굴절시켜 색을 불투명하게 만든다
물과 기름은 분자 모양과 전하가 달라서 서로 밀어내는 동시에 기름은 기름끼리 물은 물끼리 서로 잡아당긴다
때문에 섞이기 힘들고 분리되기 쉽다
계란 노른자에는 고밀도 지방 단백질(HDLs)가 저밀도 지방단백질(LDLs)에서 발견되는 인지질(레시틴)이 풍부하다
- 인지질 : 머리는 친수성, 꼬리는 소수성 - 유화제 역할 가능


마요네즈의 비율

물 1 : 기름 1 - 안정적인 유화가 되지 않고 묽으며 뿌연 액체처럼 된다
물 1 : 기름 3 - 약간 걸쭉해지고 불투명해진다
물 1 : 기름 5 = 아이스크림처럼 걸쭉해진다
- 기름을 많이 넣으면 묽어질 것 같지만 기름은 물의 입자 그물망 사이에 갇히고, 물의 움직임은 기름 때문에 제한되기 때문에 뻑뻑해진다
맛의 측면에서 기름 250ml에 달걀 1개, 물은 농도에 맞춰서 넣는 것이 좋다
기름을 천천히 넣어주는 것이 유화가 잘된다


맛이 가미된 마요네즈

마늘 마요네즈 : 마요네즈 + 다진마늘 = 아이올리라고 불리기도 한다
구운 레드페퍼 마요네즈 : 마요네즈 + 다진마늘 + 크러쉬드 레드페퍼 
시저 드레싱 : 마요네즈 + 앤초비 + 다진마늘 + 우스터소스 + 파마산 치즈 = 시저 카르디니가 만든 소스
마늘 허브 마요네즈 : 마요네즈 + 마늘 + 허브 + 차이브
홀스래디시 마요네즈 : 마요네즈 + 물기를 뺀 홀스래디시 + 디종 머스타드
치폴틀레(치폴레) 라임 마요네즈 : 마요네즈 + 라임즙 + 아도보소스에 절인 치포틀레 칠리
- 아도보 소스 : 멕시칸 칠리소스 = 할라피뇨 + 토마토 + 식초 + 허브 등
타르타르 소스 : 마요네즈 + 케이퍼 + 샬롯 + 피클 + 설탕 + 파슬리 + 소금 + 후추
선드라이 토마토 마요네즈 : 마요네즈 + 다진마늘 + 선드라이 토마토 + (파슬리)
칠리 마요네즈 : 마요네즈 + 다진마늘 + 고추장, 스리라차 등 칠리 소스
꿀 미소 마요네즈 : 마요네즈 + 미소 된장 페이스트 + 청주 + 식초 + 꿀
