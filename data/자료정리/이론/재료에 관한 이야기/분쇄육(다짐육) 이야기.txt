분쇄육

고기 덩어리가 아닌 다지거나 갈린 고기
덩어리 고기와는 원료는 같지만 특성이 매우 다르므로 따로 다루도록 한다
햄버거 : (원래는) 어떠한 조미료도 첨가하지 않은 소고기만을 갈아서 만든 패티.(소금과 후추는 겉면에만 뿌린다)
소시지 : 소금을 충분히(고기 무게의 약 1.5% - 소시지가 수분을 보존하고 좋은 식감을 만드는 역할을 한다)를 넣고
          채소나 허브, 기타 첨가물과 섞어 케이싱에 넣어 제조
미트로프, 미트볼 : 분쇄육에 빵가루, 달걀, 유제품 등을 넣고 모양을 잡아 조리하는 음식
	           달걀, 빵가루, 유제품 등이 단백질의 교차결합을 막아 고기가 더 부드럽다

고기를 직접 갈아서 쓰는 것이 좋은 이유 :
안전성(신뢰도) - 무엇이 들어갔을지 알 수 없다. 위생적 건강적 좋지 않은 결과가 발생할 수 있다
맛 - 지방 함량, 부위, 고기의 등급 등을 정확히 알 수 없어 저질의 고기를 사용하게 될 수 있다
식감 - 미리 갈아서 눌린 고기는 산화되고 퍽퍽해지고 수분이 날아가고 식감이 좋지 않다

분쇄육을 얻는 방법 : (척(소의 목과 어깨 주변 = 목심, 어깨갈비, 앞갈비, 살치살) 부위)
마트에서 산다 - 잘게 갈려있고 편리하지만 품질이 떨어질 수 있다
정육점에서 갈아주는 다짐육 - 매우 편리하고 전문성이 높아 품질이 좋을 수 있지만
			 완벽히 본인이 원하는 재료의 상태를 구현하기 어려울 수 있다
미트 그라인더로 분쇄한 다짐육 - 적당히 갈려서 식감이 좋고 편리하지만 뒤처리가 힘들다
			     소시지 고기에 적합하다
푸드 프로세서(믹서기)를 이용한 다짐육 - 크기가 균일하지 않게 갈릴 수 있고 뒤처리가 어렵지만 나쁘지 않다
칼로 다진 다짐육 - 크기가 일정하지 않아 식감이 다양하다
	           큰 패티, 두꺼운 패티에 적합하지만 하지만 매우 귀찮고 힘들고 오래 걸린다

고기 분쇄의 기본 :
좋은 고기
모든 조리기구의 저온 유지
고기가 끈적해져 성기지 않는지 확인
날카롭고 청결한 도구 사용


소시지 :

단순하면서 맛있고 인기 있는 분쇄육 음식
취향대로 만들 수 있으며 적당한 맛과 식감을 가지고 있고 꽤 많은 나트륨을 포함한다

보존처리 :
화학적 처리, 훈제, 건조/발효를 이용하여 음식을 더 오래 보관할 수 있다
이때 식감과 맛이 좋아지기도 한다
프랑스의 사퀴트리, 이탈리아의 살라미, 프로슈토 스페인의 하몽 등 - 살라미와 프로슈토는 거의 같다고 볼 수 있다
화학적 보존처리 :
소금, 설탕, 질산염, 아질산염 등을 넣어 세균 증식을 억제하고 색깔을 유지하게 한다
질산염은 고기가 숙성되면서 아질산염으로 분해된다 아질산염은 색을 보존하고 맛을 좋아지게 한다
훈제 보존처리 :
그을려진 나무는 이산화탄소를 발생시키며 이 가스는 고기 표면의 수분과 반응해 질산을 만든다
질산은 세균을 억제하며 연기 또한 지방의 산화를 막는 화합물을 만들며 나무의 재에 포함된 칼륨은 지방을 중화시킨다
질산은 또한 근육 색소의 파괴를 막는데 이로 인해 스모크링이 발생한다

건조/발효 보존법 :
세균은 수분이 없으면 살 수 없다
일부 소시지들은 야외에 매달아 말리는데 
이때 유익한 세균과 이스트가 고기를 부분적으로 분해해 좋은 맛을 만들어내기도 한다

소시지 - 살코기 + 지방 + 소금
동물이 살아있는 동안 많이 움직인 부위는 더 질기지만 맛이 좋다(대부분)
분쇄육은 어차피 부서지기 때문에 맛이 좋은 부위(돼지 목살(척), 전지 등)가 소시지에 적합하다
지방이 있어야 풍미가 좋아지며 맛 좋기로 유명한 몇몇 소시지들은 지방 함량이 20% 가까지 된다
소금은 매우 중요하다. 염도뿐 아니라 고기를 엉기게 하고 좋은 식감을 만든다
소금에 절인 고기는 수분을 더 많이 보존하며 고기를 부드럽게 한다

소시지 고기를 소금에 재우는 시간과 수분 손실량의 관계(70ºC 조리 시) : 
재우는 시간 : 0 = 수분 손실율 : 18%
재우는 시간 : 2 = 수분 손실율 : 8%
재우는 시간 : 4 = 수분 손실율 : 5%
재우는 시간 : 8 = 수분 손실율 : 4%
재우는 시간 : 24 = 수분 손실율 : 4%
so. 최소4시간 ~ 8시간 정도의 염지를 권장한다

기본적인 소시지 레시피 :
조각낸 고기를 소금에 12~24시간 냉장 염지한다(소금은 고기 무게의 1~2%)
고기를 갈고 잘 치댄 후 케이싱으로 마무리한다


미트로프 :
가끔 급식으로 나와서 먹었던 네모난 야채 박힌 고기 완자
국내에서는 별로 먹지 않고 별 관심도 없다

간단 정리 : 
돼지고기 + 소고기 + 젤라틴(수분 보존율을 높여준다) = 맛과 질감이 좋아진다
달걀의 노른자 : 맛과 수분 공급, 점성으로 안정성 증가(단백질 + 지방)
달걀의 흰자 : 노른자를 도와 느슨한 단백질 추가 (단백질 + 수분)
우유 : 연육작용은 없지만 수분과 지방을 공급해 부드럽게 느끼게 하고 맛을 좋아지게 한다
빵가루 : 고기의 단백질이 너무 단단히 결합하는 것을 막아준다
버섯 : 고기에 수분과 진한 맛을 더한다
소금은 재료 혼합 직전에 추가한다 - 고기의 점성이나 단백질의 결합이
미트로프의 경우에는 좋지 않은 현상이기 때문(햄버거 패티 또한 마찬가지)
소금은 재료 혼합 


미트볼 :
미트로프와 비슷하지만 탄성을 가지고 있다
소금을 넣고 소고기와 돼지고기를 갈아서 치대어주면 된다
- 간장, 앤초비, 달걀, 치즈, 마늘, 파슬리, 샐러리, 당근 등을 추가할 수 있으며
- 더 확장해서 빵가루, 버터밀크, *젤라틴을 추가하기도 한다

일반적으로 미트볼을 익히는 방법 :
1. 시어링  또는 프라잉
2. 보일링
1 + 2 = 브레이징이라 한다
이때 몇 개의 고기는 시어링 후 으깨서 보일링 하면 소스나 국물에 진한 맛을 더할 수 있다


버거(패티)

맛있는 버거의 조건 :
소고기를 잘 골라 직접 갈아서 바로 사용
-신뢰성과 기호성
고기를 심하게 치대지 않는다
- 간 고기는 활성화 상태이다. 반죽, 온도, 소금에 반응한다. 
- 과하게 치대면 단백질이 교차결합하여 질기고 뻑뻑해진다. 
- 또한 육즙과 치즈를 머금을 공간이 좁아진다
패티를 만들고 난 뒤에 소금을 친다. 후추는 넉넉히
- 소금이 단백질을 녹이고 교차결합하여 탄력은 만드는데 촉촉하고 부드러워야 하는 버거 패티에는 적합하지 않다
제대로 된 빵을 사용한다
- 버거에서 빵은 굉장히 중요하다. 빵을 잘못 쓰면 음식을 완전히 망칠 수 있다
센 불에서 조리한다
- 크러스트와 마이야르 반응

버거를 누르는 경우 : 
고기를 잘 익히기 위해 패티를 구울 떄 눌러줄 수 있다
하지만 굽기 시작하고 30초 이내에 이 작업을 해준다
고기를 구우면서 계속 누르게 되면 꽤 많은 육즙이 이탈하게 된다
특히 가운데 부분을 눌러주면 좋은데
고기가 익으면서 수축할 때 가운데 부분은 덜 가열되어 덜 수축하게 되어
가운데 부분만 볼록하게 올라오기 때문이다
리버스 시어링을 하게 되면 이러한 수고를 조금 덜 수 있다
전체적인 조리 시간은 리버스 시어링이 오래 걸리지만 파쿠킹(전처리)의 개념으로 
조리 시 주문을 받았을 때 서비스 시간을 단축할 수 있다
